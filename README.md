# CDT Software Test

This project was developed as a technical test for **CDT Software**, consuming the public API [JSONPlaceholder](https://jsonplaceholder.typicode.com/) and showing a list of users with details for each one.

It was built using the following technologies: React, Vite and Typescript.

## Requirements

- [Node 24.13.0 (LTS)](https://nodejs.org/en/download)

## Installation and Running
You can install the project dependencies running:
```bash
npm install
```

After that, you can run the project locally:
```bash
npm run dev
```

Then, open your browser and access:
```bash
http://localhost:5173
```
